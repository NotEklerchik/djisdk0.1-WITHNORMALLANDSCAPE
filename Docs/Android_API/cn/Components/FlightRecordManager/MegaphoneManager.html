      描述：

      开启流式传输模式。你可以调用sendRealTimeDataToMegaphone向喊话器发送需要播放的数据，调用appendEOFToRealTimeData添加EOF标志位，表示数据发送完毕。然后调用startPlay接口进行播放。
      注意：流式传输模式的数据必须为OPUS编码格式的音频数据。
