      class IUpgradeManager

      包：dji.v5.manager.interfaces
      描述：

      固件升级管理类，可通过此类获取可升级模块的固件信息。当获取到可升级提示后，你可以打开DJI官方App进行固件在线升级，也可以调用startOfflineUpgrade固件离线升级接口进行升级。

      MSDK 5.1.0开始支持
